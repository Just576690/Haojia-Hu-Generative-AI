This repository contains the work for Homework 1 of the AI-Driven Sequential Decision Making course. It demonstrates the basic workflow of using a coding assistant (Claude Code) to generate content, making manual edits, and tracking changes with Git.

## Git Workflow Summary

The following steps were used to build this repository:

1. Initialized a local Git repository with `git init`
2. Created the three required files and made the initial commit
3. Used Claude Code to generate content for `assistant_draft.md` and committed
4. Manually edited `assistant_draft.md` into `final_version.md` and committed
5. Updated `README.md` with the video link and pushed all commits to GitHub
